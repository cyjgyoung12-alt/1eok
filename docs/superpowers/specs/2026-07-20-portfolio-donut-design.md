# 자산 포트폴리오 도넛 설계

날짜: 2026-07-20 · 상태: 사용자 승인 완료 (위치: 흐름 탭 맨 위)

## 목적

계좌별 자산 비중을 원그래프(도넛)로 상시 확인. 첫 화면 4카드 원칙은 유지하고
흐름 탭 최상단에 "자산 구성" 카드를 둔다.

## 데이터

- 최신 결산의 계좌별 잔고가 기준(결산이 진실원). 카드에 "N월 결산 기준" 명시.
- `portfolioShares(balances, accounts, maxSlices=5)` 순수 함수(logic.js):
  잔고>0만, 금액 내림차순, 비중 % 계산. 계좌가 maxSlices를 넘으면 하위를
  "기타"로 합산. 삭제된 계좌의 잔고는 "(삭제된 계좌)"로 표기.

## 시각화 (의존성 없음, SVG)

- 도넛: stroke-dasharray 원호, 조각 사이 2px 간격(1조각이면 간격 없음),
  12시 방향 시작. 중앙에 총자산 숫자.
- 색: Void 단일 액센트 원칙에 따라 민트 명도 단계 램프(밝은 민트 → 어두운
  민트 → 회색). dataviz 검증 스크립트로 인접 대비 확인. 정체성은 색만으로
  구분하지 않는다 — 모든 조각은 아래 목록에 색점·이름·금액·% 직접 표기.
- 목록: 색점 · 계좌 이름 · 금액(tabular-nums) · %.
