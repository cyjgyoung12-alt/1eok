# 1억 모으기

모바일 앱처럼 쓰는 개인 자산 목표 대시보드입니다. 핵심 목적은 두 가지입니다.

- 1억까지 남은 거리와 예상 달성 시점을 직관적으로 보여주기
- 지출/수입 업데이트를 최대한 간편하게 만들기

## 실행 방법

### 가장 간단한 방법

`index.html`을 브라우저로 열면 바로 실행됩니다.

### VSCode에서 로컬 서버로 실행

MacBook 기준:

```bash
python3 -m http.server 4173
```

그 다음 브라우저에서 엽니다.

```text
http://localhost:4173
```

Node/npm을 쓰고 싶다면:

```bash
npm run dev
```

## 파일 구조

```text
index.html      앱 진입점
styles.css      모바일 앱형 UI 스타일
app.js          상태, 계산 로직, 화면 렌더링
manifest.json   PWA 기본 설정
CLAUDE.md       Claude Code 작업 지침
```

## 현재 구현된 핵심 기능

- 첫 화면: 1억까지 남은 금액, 현재 자산, 예상 달성 시점, 이번달 수입/지출
- 빠른 입력: 지출/수입 입력
- 고정 항목: 월급, 월세, 통신비 등 매월 자동 기록
- 변동자산: 주식/코인 평가액 캡쳐 업데이트 및 이력
- 속도 계산: 데이터 부족 시 월 목표 저축액 기준, 데이터가 쌓이면 최근 3개월 평균 기준
- 데이터 저장: 브라우저 `localStorage`
- 백업/복원: JSON 파일

## 중요한 제품 방향

이 앱은 일반 가계부가 아닙니다. 목표는 “거래를 꼼꼼히 정리하는 것”보다 “1억까지 남은 현실을 매번 직관적으로 느끼게 하는 것”입니다.

첫 화면은 계속 단순해야 합니다.

- 1억까지 남은 금액
- 현재 자산
- 지금 속도로 예상 달성 시점
- `+ 지출`
- `+ 수입`
- 이번달 수입/지출

상세 분석, 고정비, 변동자산, 백업, 계산 기준 설명은 첫 화면 밖에 둡니다.

## 데이터 주의

현재 데이터는 브라우저의 `localStorage`에 저장됩니다. 브라우저 데이터를 삭제하면 앱 데이터도 사라질 수 있으므로, 실제로 사용한다면 설정 탭의 백업 기능을 사용하세요.
